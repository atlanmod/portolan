Full View Extension

This extension point is the central registration mechanism of visual functionnalities inside Portolan. It is used by the default viewers provided with the Portolan application (Prefuse-based graph viewer and Modisco model browser).

Syntax

The extension point ID is: fr.inria.portolan.visualization.fullView.

The extension attributes are:

  • id: String (REQUIRED). The ID which relies a button to the corresponding view (so is also button ID).
  • commandId: String (REQUIRED). The id Eclipse will rely on to find it and execute corresponding action.
  • text: String (REQUIRED). The label for the corresponding button.
  • editorId: String (REQUIRED). The editor to use to display the subset model.
  • name: String (OPTIONAL). The viewer label.
  • pluginId: String (OPTIONAL). The Eclipse ID of the plugin which contains the icon specified below.
  • iconPath: String (OPTIONAL). The path of the icon, relative to the plugin root.

The corresponding editor class must be a subclass of the org.eclipse.ui.part.EditorPart class.

Effects

The Portolan Visualization plug-in has a specific initializer which inserts a button in the left bar of the perspective (view bar, known as VDsBar in code). For each declaration found using this extension, it tries to create the corresponding button (and underlying command) into the view bar of the Visualization perspective.

When a user clicks on such a Full View button, the corresponding command is run: it takes the referent model and opens it with the specified editor. The result is a visualization with the whole elements of the referent user model.

Example

The Portolan application comes with a model browser from MoDisco project. As a good example, this model editor extension has the parameters below:

  • fr.inria.portolan.visualization.fullView extension:
    • full_view.id: fr.inria.portolan.visualization.explorer.ExplorerView
    • full_view.commandId: fr.inria.portolan.visualization.explorer.browser.fullView
    • full_view.text: Full Model Explorer
    • full_view.editorId: org.eclipse.gmt.modisco.infra.browser.editorID
    • full_view.name: Portolan Full Model Explorer
    • full_view.pluginId: fr.inria.portolan.visualization.explorer
    • full_view.iconPath: icons/MoDiscoModelIcon.gif
