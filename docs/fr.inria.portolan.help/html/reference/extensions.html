Portolan Extension Points

The Portolan Model Cartography application can be extended using an extension mechanism based on the Eclipse extension points.

The design of Portolan enables extensions from the users: it relies on the View Definitions. See the corresponding page for additional information.

Some Portolan extension points target additions to the application through plug-ins providing new visual functionnalities. The main graph visualization and the defaut Modisco model browser also use this mechanism to provide their features into Portolan. See the plug-ins page and corresponding extensions page for more information.

The >Portolan cartography framework includes a build mechanism which takes care about changes in the user project.
