First Launch of Portolan

When you launch Portolan for the first time, its creates a Portolan directory in your user space. The Portolan framework opens on the Visualization perspective.

The workspace is empty so you get an error message if you try any of the visualization buttons on the View bar on the left.

In order to have a better idea of Portolan features, you have to install the provided example. Go in Portolan File menu, and then select New..., click on Portolan Tools Cartography Example. Once it is done, you can use the View bar buttons and see what happens.

Now, you may switch to the Edition or the Discovery perspectives and look at the structure and data of the example project.

You are now ready to create your own Portolan project.
