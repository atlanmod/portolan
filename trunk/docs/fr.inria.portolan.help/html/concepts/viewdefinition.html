View Definition

A View Definition is a specialized model transformation, which purpose is to filter the model and produce a sub-set of it, relying on the same metamodel. This transformation can compute elements to create new ones or change/add some values on existing ones, and remove those which have no interest in this view.

View definitions in Portolan are associated with a specific extension point. Users can add new views on their central model in order to visualize sub-sets of it.
